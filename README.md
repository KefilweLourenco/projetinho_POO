# 💻 Projeto POO em TypeScript

Este projeto foi desenvolvido como parte dos estudos de Programação Orientada a Objetos (POO), com foco em entender na prática conceitos fundamentais como classes, herança e encapsulamento.

## 🧠 Conceitos trabalhados

- Criação de classes
- Herança
- Encapsulamento
- Organização de código em TypeScript
- Estrutura de um sistema baseado em objetos

## ⚙️ Tecnologias utilizadas

- TypeScript
- Node.js

## 🚀 Como executar o projeto

```bash
npm install
npx ts-node src/Menu.ts
